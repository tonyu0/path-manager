# PathManager

![メイン画面](URL)

PathManagerは、頻繁にアクセスするディレクトリパスをプロジェクトや用途ごとのタブで整理し、ワンクリックでエクスプローラーや外部ツール（未実装）から開くことができる効率化ツールです。

## ✨️主な機能

### 📋タブによるカテゴリ管理
- プロジェクト別管理: ```New```メニューから新しいタブを作成し、用途に合わせてパスを分類できます。

- タブへの操作: タブのリネームや削除、並べ替えができます。

### 🪶パスへのクイックアクセス
- 起動: 各行の```Open```ボタンから、エクスプローラーや外部ツール（未実装）で対象ディレクトリを開きます。

- 検索フィルタ: 大量のパスを登録していても、検索ボックスからインクリメンタルサーチ（リアルタイム絞り込み）が可能です。

- お気に入り＆履歴: 
   - お気に入り設定（★）: 重要なパスを上位に固定表示できます。
   - 履歴管理（LastOpenedAt）: 最後に開いた日時を自動的に記録し、最近使った順のソートが可能です。

### ⚙️カスタマイズと自動保存
- 自動セーブ: アプリを閉じる際、すべてのタブ情報と設定が```appdata.json```に自動保存されます（アプリの実行ファイルと同じ場所に保存）。
- 表示設定: 「Sort by last opened」や「Show favorites first」を有効にすることで、リストの表示方法をカスタマイズできます。

## 🛠️動作環境
- Windows 10 / 11
- .NET 8.0 Runtime
